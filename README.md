# Budget RF Lab

Build a RF lab on a budget,  for homebrewers. This repository is intended to be a `Systemization of Knowledge (SoK)` set.

It is organized as below,
- Equipment list: If you'd like to contribute for your geography or language please create the appropriate markdown file. As an example we have [English version for India](en_in.md) and [Turkish version for Türkiye](tr-tr.md).
- [FLOSS Tools](FLOSS_Tools.md): Has all the tools for you to be able to use them. ([Turkish version](FLOSS_Tools_tr.md))
- [Projects](Projects.md): Has a list of projects that can get you started. ([Turkish version](Projects_tr.md))
- [RF Parts](RFParts.md): A list of useful hacker parts for RF communications projects. ([Turkish version](RFParts_tr.md))
